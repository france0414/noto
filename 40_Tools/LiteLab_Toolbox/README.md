# LiteLab Toolbox (輕量化開發工具集)

## 1) 簡介
LiteLab 是由使用者開發的一系列輕量化網頁工具，旨在解決日常開發與設計中的重複性痛點，特別是影像處理、表格轉換與色彩控制。

- **官方連結**：[https://france0414.github.io/litelab/](https://france0414.github.io/litelab/)

---

## 2) 核心工具清單

### 🖼️ Cropper (影像批次裁切)
- **功能**：
  - 支援多圖批次上傳與裁切。
  - 調整邊框圓角 (Border Radius)、內距 (Padding) 與背景色。
  - 品質控制與 WebP/PNG 格式轉換。
- **適用場景**：快速將不同比例的產品圖統一規格化。

### 🎨 Color Control (色彩控制與風格轉換)
- **功能**：
  - 色彩抽樣與對齊。
  - AI 風格轉換 (Style Transfer)，將特定視覺風格套用到目標圖片。
- **適用場景**：品牌視覺統一、UI 色調調整。

### 📊 Table Converter (表格轉換專家)
- **功能**：
  - **精準解析**：完美保留 Word/Excel 中的「合併儲存格 (Rowspan/Colspan)」。
  - **多格式輸出**：可轉換為 HTML (Odoo 友善)、CSV、JSON 等格式。
- **適用場景**：將客戶提供的複雜規格表快速轉化為網頁結構。
